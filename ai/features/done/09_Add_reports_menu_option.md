# Add Reports menu option

# Description

As an administrator I want to access to the payments/reports/
webpage.

## Requirements

1. I want to add a Menu option called **Reports** in the menu bar
2. In the Reports option, I want to have a link to the payments/reports/ webpage
   called **Payments**.

## Acceptance criteria

I can access the payments/reports/ webpage by clicking Reports -> Payments in
the main bar.
