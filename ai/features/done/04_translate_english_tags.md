# Translate English tags into Spanish

## Description

In the clients/search and admin/equipment/equipment there are English tags.

The tags must be translated into Spanish.

These are the tags per endpoint to translate:

clients/search

- Searching...
- Search clients...
- Filtered by
- Client NOT FOUND
- Start typing to search clients...
- Next
- Last

admin/equipment/equipment/

- Seleccione equipment a modificar (Just translate equipment)
- AÑADIR EQUIPMENT(Just translate EQUIPMENT)

## Acceptance criteria

- The tags in both endpoints are translated into Spanish
- The tags are added to the proper i18m files
