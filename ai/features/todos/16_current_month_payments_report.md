# Current Month Payments Report

## User story

As a system user, I want to get the payments reports grouped by date during the current month, so that I can quickly spot the earnings. Another goal is to have the input fields preselected with the proper values to query the report.

## Acceptance criteria

Given I'm on the payments reports page, When the page loads, Then the start/end date fields are pre-filled with the first and last day of the current month, the graph was automatically rendered with the values set.

## Definition of Done

Code reviewed, tested, lint passes, merged to main, no regressions.
