# Filter highlighting

## Description

In the /clients webpage, we can search for a client using his name, email, or mobile
number.

The search by name shows a highlighting in the found records that match the
data in the search filter.

The data found by email or mobile number does not have this highlighting
functionality. Due to that, we want to add this feature to the email and mobile
searches.

Use the @specs/005-add-client-search folder data as reference.
